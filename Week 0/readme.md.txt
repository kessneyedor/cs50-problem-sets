# CS50 Problem Set 0 - Scratch Project

## 🎮 Project Title
Ball and Paddle Game

## 📖 Description
This is a simple interactive game built using Scratch. A ball moves continuously across the screen, bouncing off the edges. The player controls a paddle and interacts with the ball.

When the ball touches the paddle or is clicked, it produces a "boeing" sound effect, adding an interactive and fun experience.

## 🧠 Features
- Moving ball with continuous motion
- Collision detection with paddle
- Sound effect on interaction ("boeing")
- Basic game mechanics using loops and conditions

## 🛠️ Technologies Used
- Scratch (block-based programming)

## 📂 Files
- `project.sb3` — Scratch project file

## 🚀 How to Run
1. Open Scratch (https://scratch.mit.edu)
2. Click "File" → "Load from your computer"
3. Select the `.sb3` file
4. Click the green flag to start the game
